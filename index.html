Inpainting image Detection

    This project detects image inpainting using convolutional neural networks and an ensemble model. It utilizes a dual dataset approach, combining the CIFAKE AI-Generated Synthetic Images dataset and the DeepFake Detection Challenge dataset. Training involves transfer learning, data augmentation, early stopping, and regularization methods. Models converge with binary cross-entropy loss and frozen layers. An ensemble model combines CNN predictions, optimized via hyperparameter tuning using hyperopt library. The project showcases CNNs and ensemble methods' effectiveness in inpainted region detection, contributing to image forensics and manipulation detection.

Human Detection

  • This project aims to develop a robust human detection system that leverages machine-learning techniques to accurately identify the presence of humans in images. The system utilizes advanced analysis of image properties such as shape, texture, and color to detect human features and distinguish them from the background. The proposed methodology involves a multi-stage approach, starting with a feature extraction block based on a convolutional neural network (CNN) architectures. This is followed by an encoder-decoder architecture that generates a mask output highlighting the regions containing humans. To improve accuracy, a novel combination of CNN with optimized feature extraction or attention patterns is explored. Additionally, preprocessing techniques are applied to enhance the effectiveness of feature extraction. The system provides users with confidence levels or probabilities, enabling them to make informed decisions about the presence of humans in images and videos. By achieving accurate and reliable human detection, this project contributes to various applications, including video surveillance, image forensics, and human-computer interaction.

Low Light License Plate Recognition(LPR)

    The project focuses on developing a license plate recognition (LPR) model using VGG CNN as the core architecture. The VGG model is employed for feature extraction from license plate images. The extracted features are then utilized by an SVM classifier for license plate number classification. Various preprocessing techniques like image resizing, normalization, and noise removal are applied to enhance data quality. To address low-light images, the LPR model incorporates improvements at earlier stages such as plate detection, character segmentation, and character recognition. Best practice methodologies, including hyperparameter tuning, cross-validation, and ensemble techniques, are employed to optimize the overall performance and accuracy of the LPR model.
