AWS Data Warehouse (AWS Lambda, AWS EC2, AWS S3 Bucket, AWS Cloud9, Amazon Redshift)

    This project aims to leverage AWS technologies to construct a sophisticated data pipeline specifically for processing and analyzing image data of human bodies and body parts. The focus is on facilitating a detection project that requires the handling and analysis of complex visual data. Utilizing AWS Glue, this project will automate the extraction, transformation, and loading (ETL) processes to ensure that the image data is cleaned, structured, and stored efficiently in Amazon S3. This prepared data will then be systematically loaded into Amazon Redshift, a powerful data warehousing service that allows for extensive querying and aggregation of data. The project will utilize SQL within Redshift to perform advanced data analysis, focusing on detecting and analyzing patterns and features within the image data. The use of AWS Cloud9 will provide a seamless development environment to write, run, and debug the scripts that will manage the ETL processes and data interactions. By optimizing the data pipeline, SQL queries, and leveraging the computational power of AWS, the project aims to provide a reliable and scalable infrastructure that supports complex image data analysis tasks.

Utilizing Amazon SageMaker, a fully managed service that provides every developer and data scientist with the ability to build, train, and deploy machine learning models quickly, the project embarks on creating highly accurate models tailored to the complexities of human form and motion. Camera Tracking: Human Detection (On-Premise: Yolov8, MicroPython, Stepper Motors)

    This project showcases the integration of machine learning with physical infrastructure, utilizing a comprehensive system of hardware components and machine learning models. At its core, the Raspberry Pi Pico W acts as the central microcontroller unit, managing Python scripts, Bluetooth connectivity, and communication with devices like stepper motors and a webcam. The system employs two 12V stepper motors controlled by drivers to ensure precise movements necessary for operations such as robotic arms. A standard webcam serves as a visual sensor, capturing real-time data for monitoring and object detection. Software-wise, configuration scripts set up system parameters while main operational scripts coordinate tasks and integrate sensors and actuators with machine learning algorithms for enhanced decision-making. The project leverages pretrained YOLOv models for body and head detection, demonstrating the application of real-time analytics. BLE scripts support wireless communication, and motor control scripts facilitate precise operational control. Shared variables ensure consistent synchronization across various components.

Inpainting Image Detection (On-Premise: Tensorflow, Imagenet)

    This project detects image inpainting using convolutional neural networks and an ensemble model. It utilizes a dual dataset approach, combining the CIFAKE AI-Generated Synthetic Images dataset and the DeepFake Detection Challenge dataset. Training involves transfer learning, data augmentation, early stopping, and regularization methods. Models converge with binary cross-entropy loss and frozen layers. An ensemble model combines CNN predictions, optimized via hyperparameter tuning using hyperopt library. The project showcases CNNs and ensemble methods' effectiveness in inpainted region detection, contributing to image forensics and manipulation detection.
